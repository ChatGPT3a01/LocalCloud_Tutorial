  Part 1：環境準備（12 個實作步驟）

  📂 對應 GitHub：2.4_打造AI對話API/階段一_建立AI對話功能/

  【實作 1】檢查 Python 版本
  CMD 輸入：python --version
  預期結果：Python 3.11.0 或更高

  【實作 2】檢查 pip 版本
  CMD 輸入：pip --version
  預期結果：pip 23.0.1

  【實作 3】建立工作目錄
  CMD 輸入：cd Desktop
  CMD 輸入：mkdir AI_Projects

  【實作 4】建立專案資料夾
  CMD 輸入：cd AI_Projects
  CMD 輸入：mkdir ch2_local_ai_app
  CMD 輸入：cd ch2_local_ai_app

  【實作 5】建立虛擬環境
  CMD 輸入：python -m venv venv

  【實作 6】啟動虛擬環境
  Windows CMD：venv\Scripts\activate
  Windows PowerShell：venv\Scripts\Activate.ps1
  預期結果：提示符前面出現 (venv)

  【實作 7】安裝 Flask
  CMD 輸入：pip install flask

  【實作 8】安裝 OpenAI SDK
  CMD 輸入：pip install openai

  【實作 9】安裝環境變數工具
  CMD 輸入：pip install python-dotenv

  【實作 10】確認套件安裝
  CMD 輸入：pip list
  預期結果：看到 flask、openai、python-dotenv

  【實作 11】建立 .env 檔案
  建立檔案：.env
  內容：OPENAI_API_KEY=sk-proj-你的API金鑰

  【實作 12】建立 .gitignore 檔案
  建立檔案：.gitignore
  內容：
  venv/
  .env
  __pycache__/

  ✅ 完成檢查：(venv) 提示符 + pip list 顯示套件

  Part 4：Postman 測試 API（4 個實作步驟）

  📂 對應 GitHub：2.4_打造AI對話API/階段一_建立AI對話功能/

  【實作 1】下載並開啟 Postman
  網址：https://www.postman.com/downloads/
  💡 可點「Skip and go to the app」跳過登入

  【實作 2】建立 POST 請求
  1. 點 New → HTTP Request
  2. GET 改為 POST
  3. URL 輸入：http://localhost:5000/chat

  【實作 3】設定 Body（最重要！）
  1. 點 Body
  2. 選 raw
  3. 右側選 JSON
  4. 輸入：
  {
      "message": "你好，請介紹一下你自己"
  }
  🟥 必須要有 { } 和英文雙引號！

  【實作 4】發送請求
  1. 點 Send
  2. 等待幾秒
  預期結果：200 OK
  {
      "success": true,
      "message": "你好！我是 AI 助手..."
  }

  ✅ 完成檢查：Postman 顯示 200 OK + AI 回應
